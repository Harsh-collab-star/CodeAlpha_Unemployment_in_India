# CodeAlpha_Unemployment_in_India

Project Title: Unemployment Analysis with Python

Description:
This project analyzes unemployment data in India using Python to study
trends, Covid-19 impact, and seasonal patterns.

Technologies Used:
- Python
- Pandas
- Matplotlib
- Seaborn

Key Findings:
- Covid-19 led to a major unemployment spike
- Seasonal trends are visible
- Regional variations exist

Outcome:
Insights that can support economic and employment policies.
